# Arbitrary-Precision-Calculator-APC-
Developed an Arbitrary Precision Calculator (APC) in C capable of performing arithmetic operations on integers of virtually unlimited size, overcoming the limitations of built-in data types. The project utilizes linked list data structures to represent and manipulate large numbers digit by digit.

Key Features:
- Addition of large integers
- Subtraction of large integers
- Multiplication of large integers
- Division of large integers
- Support for numbers exceeding standard data type limits
- Linked list based number representation
- Efficient carry and borrow handling
- Dynamic memory management
- User-friendly command-line interface

Technologies Used:
- C Programming
- Linked Lists
- Dynamic Memory Allocation
- Data Structures and Algorithms
- Linux Development Environment

Learning Outcomes:
- Understanding of arbitrary precision arithmetic
- Advanced linked list manipulation
- Memory management techniques
- Mathematical algorithm implementation
- Handling large-scale numerical computations
- Software modularization and debugging

This project demonstrates the implementation of high-precision arithmetic operations by representing large numbers using dynamic data structures, enabling calculations beyond the range of conventional integer data types.
